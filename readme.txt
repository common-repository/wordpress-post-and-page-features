Embed a set of features in your Post or Page.

== Description ==
This Plugin allows you to embed a set of features in your Post or Page. It's lightweight, not too much code and css for flexible use. 

== Installation ==
1. Plugins > Add New > Search `WordPress Post and Page Features`
2. Install Now > Activate

Please see [WPTF - Post and Page Features Guide](http://www.trusted-freelancer.com/freebies/wordpress/wptf-post-and-page-features/) for the complete guide.

**Important**

Always try at your local website first.
